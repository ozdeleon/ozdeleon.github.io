Huffpost

2019 • News decoded campaign • California

Team

  • Oscar DeLeon, Designer
  • Cary Codi, Producer

Information decoded


Created an evergreen campaign for Huffpost which can be seen in the United States and the UK.

Goal

The goal of this campaign was to construct imagery that conveys multiple ideas and would pair with different messaging for A/B testing. The secondary priority was to drive app downloads.

Challenge

• Only using Huffpost signature green and black as primary colors

• Only using imagery that would couple with evolving messaging

Aproach

The exciting part of working on this campaign was that I was free to test different looks from the beginning. I started by using photography taken from Huffpost, but found that by the time it was presented to the stakeholders, the image was no longer applicable.

I knew I needed to conceive an evergreen image

So I guided the conversation away from using Huffpost’s editorial photography since news is a daily occurrence and some images might be irrelevant by the end of the day. Then I moved the focus to the main message of our campaign which was “Information Decoded”.
